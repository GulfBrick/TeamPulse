Connect to your server
Enter the URL your admin gave you. This is where TeamPulse is hosted.
Where do I find this?

Your admin will share the server URL with you. It usually looks like:

https://yourcompany.up.railway.app

It's the same URL you use to open TeamPulse in your browser.

Sign in to your account
Use a setup code from the web app, or sign in with your email.
How to get a setup code

1. Log in to TeamPulse in your browser

2. You'll see a setup code on screen (6 characters)

3. Paste it below — no password needed!

Use your TeamPulse credentials

The same email and password you use to log in on the web.
